# Personal Portfolio

A modern, responsive portfolio website built with Next.js, TypeScript, and Tailwind CSS.

**[View Portfolio](https://mikeandersonwwt.github.io/portfolio/)**

**[View Technical Details](./TECH_DETAILS.md)** - Complete documentation of technologies, features, and architecture

### View locally

Run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
